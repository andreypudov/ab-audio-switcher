* Overview
ab-audio-switcher is a tiny cross-platform tool for instant A/B audio comparison with a single hotkey.
It loads two audio tracks into memory, plays them through a real-time callback, and switches between them instantly—ideal for critical listening, codec evaluation, mix comparison, and headphone testing.


* Features
- Instant A/B switching with one hotkey
- Gapless playback using a continuous audio callback
- Raw PCM pipeline for transparent audio comparison
- Works on macOS, Linux, and Windows
- Minimal dependencies and simple architecture

* Optional: Convert audio sources to WAV using FFmpeg
If your original files are in formats like MP3 or FLAC, convert them to WAV for consistent decoding.

Example input files:
- =source_a.mp3=
- =source_b.flac=

Convert them:

#+begin_src bash
ffmpeg -i source_a.mp3 A.wav
ffmpeg -i source_b.flac B.wav
#+end_src

This produces =A.wav= and =B.wav= for use with the switcher.

* Usage
Place two audio files in the project directory:

- =A.wav=
- =B.wav=

Run the switcher:

#+begin_src bash
python ab_switcher.py
#+end_src

Press =SPACE= to toggle between Track A and Track B.

* How It Works
The program:
1. Decodes both audio files into raw PCM arrays.
2. Starts a continuous audio callback using =sounddevice=.
3. Maintains an index pointer into the active track.
4. Switches the active buffer instantly when the hotkey is pressed.

This avoids restarting the audio stream and ensures seamless switching.

* Audio Preparation Steps for Codec Comparison

To perform a fair FLAC vs MP3 comparison, both files must be decoded into the same
uncompressed PCM format. Follow these steps:

1. Convert the original FLAC into WAV (reference PCM):

#+begin_src bash
ffmpeg -i source.flac -c:a pcm_f32le A.wav
#+end_src

2. Convert the original FLAC into MP3 at a chosen bitrate (example: 146 kbps):

#+begin_src bash
ffmpeg -i source.flac -b:a 146k source.mp3
#+end_src

3. Decode the MP3 back into WAV (lossy PCM):

#+begin_src bash
ffmpeg -i source.mp3 -c:a pcm_f32le B.wav
#+end_src

You now have:

- =A.wav= — decoded FLAC (reference)
- =B.wav= — decoded MP3 (lossy version)

Both WAV files share identical format (PCM float, 44.1 kHz, stereo), making them
ideal for accurate A/B switching and analysis.
